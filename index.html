    ⚙️ 구글시트 백업 설정

    구글시트 자동 백업을 사용하려면 Google Apps Script 웹앱 URL이 필요합니다.

    1. 새 구글시트 생성 (예: "발주서 백업")
    2. 확장 프로그램 → Apps Script 클릭
    3. 아래 코드를 붙여넣고 저장
    4. 배포 → 새 배포 → 유형: 웹 앱 → 액세스: 모든 사용자 → 배포
    5. 생성된 웹앱 URL을 아래에 붙여넣기
    ⚠️ 코드 업데이트 안내: 이전에 설정하신 분도 아래 코드로 다시 교체한 뒤 Apps Script에서 배포 → 배포 관리 → 연필(편집) 아이콘 → 버전: 새 버전 → 배포를 눌러 재배포해주세요. (URL은 그대로 유지됨)
